# AL-WebSolution
## Installation

1. Crear un ambiente virtual

```bash
py -3 -m venv .venv
```

2. Activar ambiente virtual

```bash
.venv\Scripts\activate
```

3. Instalar Flask
```bash
pip install Flask
```

4. Instalar python-dotenv

```bash
pip install python-dotenv
```
